Data Types

Dynamic Variables

Null and Undefined

null is an object in Js (can be considered as a bug). undefined means a variable without a value. Its type is undefined.

We can empty an object with both x = null and x = undefined. In the former the type of x will be object, and in the latter type will be undefined.

null == undefined is true, null === undefined is false. They both mean 'empty value' but their types are different

Primitive Data Types

Primitive data type is a simple data value without any method/property.
  • Number
  • String
  • Boolean
  • Undefined

Complex Data Types

Complex data types have methods or properties.
  • Function (functions)
  • Object (object, array, null)
